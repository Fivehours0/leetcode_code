将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 


示例：

    输入：1->2->4, 1->3->4
    输出：1->1->2->3->4->4

算法(详见官方解答)  

    双“指针”
    
    l1, l2分别指向两个链表的表头，res为结果链表的头节点
    1和1相等 res->1, l2 = l2.next
    1和3比较 res->1->1, l1 = l1.next
    2和3比较 res->1->1->2 l1 = l1.next
    以此类推
